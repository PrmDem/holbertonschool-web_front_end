# Developer Tools
DevTools allow web developers to edit pages and diagnose problems easily. It's another way to check the reliability of the websites we build, with a quick access to performance, resources, or network information.

## Requirements
* Have a README.md file at the root of the folder of the project
* Use the newest version of Google Chrome browser (78.0.3904.70 or later)
* Screenshots can be taken via the OS. They're here to see how and where we use the DevTools.
